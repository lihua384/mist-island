# 🏝️ 迷雾之岛 (Mist Island)

欢迎来到《迷雾之岛》！这是一款基于 HTML/JS 开发的网页解谜小游戏。你需要在一个神秘的岛屿上探索，通过观察和逻辑推理来解开重重谜题，最终逃离这里。

## 🎮 游戏玩法

1. **探索与收集**：点击屏幕上的物品进行交互，寻找隐藏的道具。
2. **解谜与破译**：利用收集到的线索（如数字、符号、图案），破解机关或密码锁。
3. **通关目标**：找到关键道具，打开通往自由的出口。

## 🛠️ 技术栈

- **HTML5**: 构建游戏界面结构。
- **CSS3**: 负责页面样式与布局。
- **JavaScript**: 处理游戏逻辑、点击事件及谜题判定。

## 🚀 如何开始

1. 下载本仓库的所有文件到本地。
2. 双击打开 迷雾之岛.html
3. 在浏览器中即可直接游玩，无需安装任何插件。
